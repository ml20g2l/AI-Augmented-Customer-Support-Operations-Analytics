# Data Visualisation Tool Recommendation

## Recommendation

Use **Power BI Desktop** for the final dashboard in this project. It is already installed, it is free on Windows, and it demonstrates data import, modelling, DAX, interactive report design, and portfolio-ready business communication in one file.

## Alternatives

| Tool | Fastest use case | Strength | Limitation for this project | Recommendation |
| --- | --- | --- | --- | --- |
| Power BI Desktop | Business dashboard with CSV data | DAX, modelling, slicers, report pages, strong business-analytics signal | Initial learning curve | Use as the final deliverable |
| Tableau Public | Public visual storytelling portfolio | Polished visuals and a public profile | Published work is public and the workflow is less aligned with the Power BI skill target | Use only as a later optional version |
| Looker Studio | Very fast browser-based reporting | Free, quick to share, simple controls | Weaker local-file and data-modelling workflow for this portfolio | Use for Google Analytics projects later |
| Streamlit | Python-led data application | Fast for code-first exploration | Creates an app, not a conventional BI dashboard; adds deployment work | Do not use for this project |

## Practical Decision

Do not switch tools for this dashboard. The fastest path to a credible portfolio result is to build Page 1 in Power BI now using the three prepared CSV files, export a PDF or PNG screenshot, and later add the two AI pages when Free Tier validation data becomes available.
